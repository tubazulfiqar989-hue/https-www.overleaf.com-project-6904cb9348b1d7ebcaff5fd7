Purpose of Assignment
The primary objective of this assignment is to document and explain a standard Git workflow using the professional typesetting system, LaTex. The overall objective of this assignment is to document and explain a Git workflow while demonstrating version control skills
This project serves a dual purpose:
1.  To demonstrate proficiency in Git version control by tracking the entire document creation process (initialization, branching, committing, and merging).\\
2.  To showcase the ability to create a structured, high-quality document in LaTex , which serves as the final documentation of the Git workflow, including explanations and a reference table of key commands.
